Использование эмулятора терминала xterm
Настройка Xterm

xterm является стандартным эмулятором терминала для среды X Window System в Unix.

X Window System — оконная система, обеспечивающая стандартные инструменты и протоколы для построения графического интерфейса пользователя. Используется в UNIX-подобных ОС.

xterm — это полный пакет эмуляции терминала, но для работы ему не требуется много ресурсов (например, памяти). Из-за этого пакет xterm по-прежнему популярен в дистрибутивах Linux, предназначенных для работы на старом оборудовании.

Пользователь имеет возможность работать с несколькими xterm терминалами, запущенными в одно и то же время на одном и том же дисплее. Каждый из виртуальных терминалов предоставляет независимый ввод-вывод для процессов, запущенных в каждом из них (обычно это Unix shell процессы).

Интерфейс
По умолчанию у xterm отсутствует строка меню. Для того, чтобы получить доступ к одному из трёх меню xterm, пользователь должен, удерживая клавишу Control, нажать левую, среднюю или правую клавишу мыши. Поддержку строки меню можно задать во время компиляции программы; в этом случае указанные выше меню, будут доступны из строки меню.

Установил: sudo apt install xterm

Шрифты должны быть фиксированной ширины и, если указаны как обычный, так и полужирный шрифты, должны иметь одинаковый размер. Если указан только обычный шрифт, он будет использоваться как для обычного, так и для полужирного текста (путем переопределения). Параметр -e, если он указан, должен отображаться в конце командной строки, в противном случае будет запущена командная оболочка пользователя по умолчанию. Параметры, начинающиеся со знака плюс (+), восстанавливают значение по умолчанию.

Опции
Если параметр начинается с "+" вместо "-", параметру возвращается значение по умолчанию. Параметры -version и -help интерпретируются, даже если xterm не может открыть отображение, и вместе с параметром -class проверяются перед всеми другими параметрами.
При запуске xterm обычно проверяет переменную среды shell (оболочки), для какой shell запускаться. Однако эту оболочку можно изменить, указав параметр для xterm после всех других параметров, который указывает путь к запускаемой оболочке. Это имя пути может быть абсолютным путем или относительным путем, и xterm будет искать указанную оболочку в переменной окружения пользователя PATH, если не сможет ее найти. Если оболочка не указана, а переменная окружения SHELL не определена, xterm использует оболочку Bourne, /bin/sh.

Xpra

Утилита xpra — инструмент, который запускает программы X11, обычно на удаленном хосте (клиенте), и направляет их отображение на локальный компьютер (сервер) без потери состояния. Предоставляет удаленный доступ как отдельным приложениям, так и новым или существующим сеансам рабочего стола.

На сервере утилита xpra в режиме демона запускает программу с заданным идентификатором сеанса, а на клиенте происходит присоединение к этому сеансу с идентификатором.

Доступ к сеансам можно получить по SSH или через защищенные паролем обычные TCP-сокеты с SSL или без него.

Xvfb

Xvfb или X virtual framebuffer - это сервер отображения, реализующий протокол X11 display server. В отличие от других серверов отображения, Xvfb выполняет все графические операции в виртуальной памяти, не выводя никаких выходных данных на экран. С точки зрения клиента, он действует точно так же, как любой другой сервер X display, обслуживая запросы и отправляя события и ошибки соответствующим образом.

В X Window System, Xvfb — это X11 сервер, который выполняет все графические операции в памяти без вывода на экран. С точки зрения клиента он действует так же, как любой другой сервер, обслуживающий запросы и отправки события и ошибки по мере необходимости.

Установка: sudo apt-get install xvfb

X11-приложения

Xorg или X11

Проект X.Org предоставляет свободную реализацию X Window System с открытым исходным кодом.
Xorg (обычно называемый просто X) очень популярен среди пользователей Linux, что привело к тому, что большинство приложений с графическим интерфейсом используют X11, из-за этого Xorg доступен в большинстве дистрибутивов.
